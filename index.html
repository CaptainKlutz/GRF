GRF is a Recommendation Forum

Welcome to GRF, click on a tab for recommendations.

Notes

The easiest way to add these GRFs to your game is to download the example scenario provided for each preset.

Trying to load the scenario will then work with the online content downloader to automatically fetch the content you're missing.

If one doesn't exist, open your openttd.cfg, and paste whatever's in the spoiler box you want on a new line above the line beginning [newgrf].

The game will then tell you if you're missing anything when you try to load the preset in-game.

For 64+ GRFs at once to be loaded, you MUST use a patch pack such as JGRPP.

'CD' column denotes if a GRF is available on the Online Content Downloader, serviced by Bananas. If it's not available, it should be linked in Notes.

If you want to add your own list, visit the Github to start contributing.
